    Quick Troubleshooting

    App not detecting position? 1. Check sensor screen for live values 2. Verify detection modes are enabled 3. Grant Do Not Disturb permission 4. Restart the app

    Battery draining fast? 1. Enable Power Saver mode 2. Disable unused detection modes 3. Check for battery optimization settings

    Notifications not showing? 1. Grant notification permission 2. Check system notification settings 3. Ensure app isn't in Do Not Disturb

    Pocket detection not working? 1. Verify proximity sensor values change 2. Remove thick cases/screen protectors 3. Ensure phone is face-down 4. Test with fallback face-down detection
